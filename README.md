# Discord VCロガー

DiscordのVCに入退出したときに、ログを吐きます。

## 使い方

> 1	`token`ファイルにDiscordBotのトークンを書き込む<br>
2	`main.py`の`<サーバーID>`と`<テキストID>`を変更<br>
3	実行
